## Local preview

Open index.html directly in a browser, or run a tiny HTTP server from the repository root:

- python3 -m http.server 8000
- npx serve .

Then visit http://localhost:8000 in your browser.

Note: the contact form (Formspree) is live and posts to form ID `mlgqdpll`.
